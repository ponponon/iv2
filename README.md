# iv2

[简体中文](./README.zh-CN.md) | [English](./README.md)

## Introduction

Converting images to vectors for image search and image similarity comparison

> vector latitude of 512

## Dependencies

Python interpreter.

- CPython : version 3.8 and above

Installed by.

```shell
pip install iv2
```
